# SaaS Dashboard (React + Firebase)

🔗 Live Demo: https://saas-pms-dashboard-sc.web.app

A modern SaaS admin dashboard built with React and Firebase, featuring authentication, real-time data, and project management.

## Features

- Firebase Authentication (Login/Register)
- Protected Routes
- Real-time Projects CRUD (Firestore)
- Dashboard with live data
- Settings page (profile & security)
- Toast notifications & loaders

## Tech stack

- React (Vite)
- Firebase (Auth, Firestore)
- Bootstrap + Bootstrap Icons
